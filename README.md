# 学生成绩管理系统 python731

## 项目概述

学生成绩管理系统是基于Python语言和Django框架开发的应用，旨在帮助学校管理学生成绩相关事务。

## 技术栈

- **后端语言与框架**：Python 3.x，Django
- **数据库**：MySQL
- **开发工具**：PyCharm

## 功能模块

- 成绩录入与查询
- 学生信息管理
- 课程信息管理
- 成绩统计分析

## 系统角色

- 管理员：负责系统的维护与用户管理
- 教师：进行成绩录入与学生信息查询
- 学生：查询个人成绩

## 运行环境

- 操作系统：支持Linux与Windows
- Python版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 确保Python 3.x和MySQL环境已正确安装
2. 克隆或下载项目代码到本地
3. 使用PyCharm或其他IDE打开项目，并配置数据库连接
4. 执行数据库迁移命令，建立数据库表结构
5. 运行项目，并根据指引完成后续配置
